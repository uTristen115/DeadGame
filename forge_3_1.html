FORGE CONTROLS

Camera
RMB drag to orbit   MMB or Shift+RMB drag to pan   wheel zoom
WASD move camera target   F focus on selection
Placing
Pick an asset, move mouse, LMB to place (keeps placing — Esc to stop)
R / Shift+R rotate the placement ghost or selected objects/walls ±15°   T rotate 90°
Q / E or Alt+wheel lower / raise placement height
With STACK on, props and walls placed on top of walls/platforms/stairs get the right height; Q/E adds on top of that
Interactive
Signs render any text · Buyable Doors use "gap along" (x or z) to match the wall they sit in — leave a ~3-wide gap in that wall run
Weapon Wall-Buys choose a weapon id from the dropdown · place only one Power Switch
Script Buttons trigger custom events. Reuse an event id on multiple buttons, then set required to 0 for all buttons or a number for a partial sequence.
Player Starts set where players spawn and which way they face — the wedge is their first view. Place as many as you need. Saved as "spawn" for solo compatibility and "spawns" for multiplayer.
Room Zones are blue rectangles. Give each room a simple id like lobby, hall1, storage. Use outside for the exterior.
Nav Portals are yellow connectors for open paths, stairs, and non-door openings. Buyable Doors and Boarded Windows also have zone fields and become graph links automatically.
Zombie Spawns are red markers. Assign each one to the zone it physically sits in. The host only uses spawns with an open path to a player.
Active zones are zones with a player in them, or zones a player was in recently. The host prefers boarded windows attached to or near active zones, spawning zombies on the outside pad.
Boarded Windows: the red SPAWN cone marks where zombies appear, the green pad where they climb in — flip "spawn side" to swap. Set outside/room zones so pathing knows what they connect.
Walls
WALL mode: LMB sets the start, move, LMB again to finish. Runs are axis-locked like the game's addWall. Q/E raises or lowers the next wall run. Esc cancels.
Editing
LMB select · Shift/Ctrl+LMB add/remove · drag selected objects to move · Q/E raise/lower
Use the floating move, rotate, and scale icons to transform one object or the entire multi-selection directly in the workspace.
Del delete   Ctrl+D duplicate   Ctrl+Z undo
Saving
EDIT HARVEST loads an editable Forge conversion of the original Harvest Facility. Save it as JSON, then use the game's LOAD MAP .JSON option to test your edited version.
SAVE/LOAD .json keeps your layout. EXPORT CODE generates paste-ready JS for game.html. Work also autosaves to this browser.
